# Portfolio IA Aplicada

**Formación, Riesgo y Soluciones Inteligentes**

Portfolio profesional orientado a mostrar proyectos de inteligencia artificial aplicada, formación, documentación inteligente, análisis estratégico y comercio internacional.

## Objetivo

Este repositorio contiene la web principal del portfolio. Su función es actuar como punto central de presentación, navegación y acceso a los distintos proyectos que componen el ecosistema del portfolio.

El enfoque del portfolio no se limita a mostrar código o ejercicios técnicos aislados. Está diseñado para reflejar una manera de trabajar la IA desde una perspectiva aplicada, útil y profesional, conectada con docencia, consultoría, documentación, riesgo y análisis internacional.

## Estructura de la web

La web principal incluye las siguientes páginas:

- `index.html` — portada y presentación general
- `proyectos.html` — visión general de los proyectos del portfolio
- `formacion-docencia.html` — línea de trabajo vinculada a formación, aprendizaje y diseño pedagógico
- `comercio-analisis.html` — especialización en riesgo país, inteligencia competitiva y análisis internacional
- `sobre-mi.html` — perfil profesional y trayectoria
- `contacto.html` — página de contacto y estado del portfolio

## Proyectos principales del portfolio

- **AI Readiness & Strategy Hub**  
  Diagnóstico, madurez y hoja de ruta para adopción de IA.

- **GlobalRisk AI**  
  Análisis de riesgo país y entorno estratégico para comercio internacional.

- **LicitIA — RAG Documental**  
  Consulta inteligente sobre normativa, pliegos y documentación compleja.

- **Academy AI**  
  Programas formativos, capacitación ejecutiva y tutor IA.

- **EduCraft AI**  
  Diseño de experiencias de aprendizaje, actividades, rúbricas y evaluación.

- **CompeteMap**  
  Inteligencia competitiva, análisis de mercados y apoyo a exportación.
